# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to abide by its terms.

To report a violation, contact a project maintainer privately — for example, via [GitHub's private reporting tools](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) or by emailing the maintainer directly. Do not open a public GitHub issue to report conduct violations, as this may expose sensitive information.
